# 帕拉塞尔苏斯的三原质 / Paracelsus's Tria Prima

## 是什么
万物不过盐、硫、汞的三重奏——肉体之盐、灵魂之硫、精神之汞——疾病不是体内的败坏，而是体外的入侵者。

## 可借元素
**性相映射：** 杯（盐——固定的肉体，血液与骨头的盐分，生命的载体）、铸（硫——燃烧的灵魂，一切变化的驱动力）、蛾（汞——挥发不定的精神，既可以是毒也可以是药，一切形态的逃逸者）。帕拉塞尔苏斯的革命在医学史上无人不知——他烧了盖伦的医书，宣布疾病不是体液失衡，而是来自外部的"存在体"（ens）入侵——这五个ens（星辰之ens、毒物之ens、自然之ens、精神之ens、上帝之ens）本身就是五种密教性相的原型。

**适用实体类型：** 仪式 / 原料 / 影响 / 追随者 / 典籍。

**具体用法：**
1. **原料体系**——"盐""硫""汞"作为三种可获取的常见原料：盐（Nether Salt）来自地下矿脉或在仪式中从液体中蒸发得到，具有杯+冬的双性相；硫（Brimstone of the Soul）来自火山之地或活体蒸馏，具有铸+灯的双性相；汞（Quicksilver Spirit）来自活的植物蒸馏液或低温升华产物，具有蛾+启的双性相。三种原料合在一起，可以将任意追随者转化为"哲人"——一个失去了所有个性但获得全性相不死性的存在。
2. **仪式卡**——"签名之术"（Doctrine of Signatures）：选择你拥有的任意一种植物类原料（如药草），将其属性"转移"到追随者身上——因为每一种植物的形状都标注了它治疗的人体器官。例如一枚外形像肺的叶子可以治愈追随者的呼吸类诅咒，一片像瞳孔的花可以增强追随者的灯之视野。
3. **影响卡**——"五重Ens"：场上同时存在五种ens——星辰ens（灯）、毒物ens（杯）、自然ens（刃）、精神ens（冬）、上帝ens（启）。每获得一种，你的总部状态随之改变。集齐五种时，你将获得一次创造"人造人"的机会——见下方。
4. **追随者卡**——"霍尔蒙库鲁斯"（Homunculus）：通过一个特殊的四十周仪式制作。原料：密封玻璃容器 + 人血（一份杯之原料）+ 马粪的温热 + 每日必定执行一次的维护行动。四十周（四季）后，孵化出一个没有面孔但具有极高某种性相的微型人。它占据追随者槽位，但不需要薪酬，不产生恐惧，不会被猎手审问——因为它没有法律意义上的"身份"。但它会质疑自己为什么存在，每季进行一次"存在之问"，如果玩家连续三次无法给出满意的答案（消耗一份秘史或灯类知识），它将自毁。
5. **典籍**——"帕拉格拉农"（Paragranum）：阅读后解锁"签名"知识——从此你对所有自然类原料（药草、矿物、昆虫、动物部件）不再需要辨识，直接可以看到它们的性相归属。

**关键意象：** 坩埚中沸腾的液体分离为三层：底部结晶的白盐、中部燃烧的黄硫、顶部蒸发的银汞；植物叶脉中浮现出人眼的形状；密封容器中悬浮着手指大小的人形；烧医书的火焰在广场上被点燃。

## 来源
帕拉塞尔苏斯（Paracelsus, 约1493-1541），本名菲利普斯·奥雷奥卢斯·特奥弗拉斯图斯·博姆巴斯特斯·冯·霍恩海姆（Philippus Aureolus Theophrastus Bombastus von Hohenheim）。瑞士-德意志医生、炼金术士、神学家，文艺复兴"医疗革命"的旗手。1527年在巴塞尔大学公开烧毁阿维森纳和盖伦的医书以宣布旧医学的死亡。主要著作包括《帕拉格拉农》《帕拉米鲁姆》《论矿工的疾病》等。他的三原质理论（盐-硫-汞替代了古希腊的四元素说的炼金地位）远不止是化学创新——它本质上是将人的身心灵结构投射到一切物质中。
