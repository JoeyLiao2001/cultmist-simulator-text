# 桃木剑

**是什么** 桃木自古为"五木之精"，能压伏邪气，慑服百鬼。桃木削制的剑——道士步罡踏斗时挥舞之器——不斩肉身，专破妖邪。一剑劈下，鬼魅现形；一剑刺出，煞气溃散。

**可借元素**
- **密教准则**：刃（斩杀、破邪）与心（镇守、护佑）。桃木剑不是寻常的"刃"——它的力量源于生长（春天、桃树开花），因而以生机对抗死气。
- **适合实体类型**：武器 / 防具 / 仪式工具。可设计为：装备桃木剑对抗蛾类、冬类实体时有伤害加成；放置在探索栏作为"护符"减少遭遇负面事件的概率。桃木剑不可重复附魔但耐久消耗后可用"桃木"原料修复。
- **命名建议**：桃木剑 → "辟邪木剑""三罡法剑""春杀之刃"。桃符（对联的前身）可作为低级消耗品版本，桃木剑为高级永久版本。
- **具体意象**：桃枝插门——最简单的防护；神荼郁垒二神立桃树下阅鬼——桃树的惩戒功能源于此；剑柄刻七星或雷纹以增强效力。

**来源** 《论衡》引《山海经》佚文、王充《论衡·订鬼》篇。
