# 欧米伽点 / Omega Point

## 是什么
进化不是随机的偶然——所有物质、生命、意识都在向一个终极收束点趋近，那个点就是"基督意识"。

## 可借元素
**性相映射：** 灯（趋同的光——所有的射线都来自同一个光源，又汇聚于同一个终点）、铸（物质性本身的转化——从惰性质子到活细胞到自我意识，铸造出一条不可逆的上升链）、启（欧米伽点在当下的吸引力——它尚未抵达，但它的引力已经在拉动现在——这正是启相的本质：未来的知识牵引当下的行动）。德日进的独特之处在于：他把科学进化论和基督教终末论熔为一炉，形成了一种同时满足理性与神秘感的"演化神秘主义"。

**适用实体类型：** 世界状态 / 结局条件 / 影响力机制。

**具体用法：**
1. **引力机制**——"终点的拉力"：在游戏开始时，生成一个隐藏的"欧米伽距离"数值。该数值表示你当前状态与欧米伽点（完美结局）之间的差距。你每次做出任何提升性相、获取知识、升级追随者的行动，该数值都会微幅下降。但关键在于——数值不可见。你只能通过一个特殊行动"感知引力"来间接探测它（行动描述为："你感到一种不可名状的牵引——仿佛你的一切行为都被某个遥远的、尚未抵达的未来在吸取"）。当数值降到零时，自动触发最终结局。
2. **智力圈/灵生圈**——"集体意识场"：当你拥有足够多的高阶追随者（特别是那些专精灯、启、铸的），一个隐藏的世界属性"智力圈强度"开始上升。智识圈强度每提高一级，所有追随者的学习速度提升一级——因为这象征一个集体意识场的形成：思想不再孤立，一个密修者的顿悟会以不可见的共振加速他人的理解。
3. **物质-意识转化**——"复构性"：德日进提出物质有一个内部（意识）和外部（形式）。你可以在铸相工作台上发明一个特殊原料操作——"内部翻转"：将任何物质性原料（金属、石料、木材）翻转为其"内部"（意识侧面），获得一份启相或灯相灵质。
4. **欧米伽结局**——"点与终点的重合"：触发后，结局文本不以你升入漫宿或成为司辰结尾，而是"你的意识进入了一个引力场的收束轨道——你不再向外扩展，而是向内凝聚。你发现你所见的一切——漫宿、司辰、辉光，乃至于你自身——都只是同一个真实在不同阶段中的折射。在最终的收敛点上，所有折射重合为一个无限密度的光点。那个光点没有名字，但你觉得它非常、非常面熟。"

**关键意象：** 一个呈现为螺旋的引力场——时间线不是直线，而是纺锤形的——从阿尔法（物质的开端）散射开来，再向欧米伽（意识的终结）聚拢；一条鱼离开海洋爬上陆地、一只爬行动物的前肢变成翅膀、一个智人在洞穴中点燃第一把火、一个修士在实验室中看着显微镜下细胞分裂——它们都是同一个轨迹上的不同的点。

## 来源
皮埃尔·泰亚尔·德·日进（Pierre Teilhard de Chardin，1881-1955），法国耶稣会神父、古生物学家、地质学家。他的核心著作《人的现象》（Le Phénomène Humain）提出：宇宙存在一个"内部性"维度（物质从起分子就有一个内在的朝向更复杂和更多意识的趋向，他称之为"复杂性-意识律"），人类出现后产生"智力圈"（noosphère——思想的包裹层），最终整个宇宙通过基督-欧米伽点收束于一个"超人格的最高中心"。他的思想生前被教会禁止出版，死后成为影响深远的神学-科学跨界叙事。
