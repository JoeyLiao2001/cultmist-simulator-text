# 符箓

**是什么** 以朱砂黄纸书写、内蕴神力的秘文字。笔画曲折如云纹雷纹，非读之文字，乃用之令咒——书写即召请，焚烧即发令，佩戴即护身。符者，天地之信。

**可借元素**
- **密教准则**：启（天启、洞开）与灯（照亮、驱散）。符箓是"书写即生效"的范式——一旦完成笔画，世界已被改变。
- **适合实体类型**：可书写可消耗的工具卡 / 一次性仪式 / 防护机制。可设计为：一张空符纸 + 特定秘传 = 各种效果的符箓（镇邪符、召将符、隐身符、五雷符）。符箓折叠后放置在不同工作槽触发不同效果。
- **命名建议**：符箓 → "灵宝真文""云篆天书""赤书玉字"。可设置"符头（敕令）—符胆（神名）—符脚（结煞）"三段式结构，每段决定一种效果。
- **具体意象**：朱砂如血、符纸为黄、火焚时青烟直上达于神庭；画错了符头会自燃。

**来源** 东汉张道陵天师道所传，后世灵宝派、神霄派各有体系。
