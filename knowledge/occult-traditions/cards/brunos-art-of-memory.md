# 布鲁诺的记忆术 / Giordano Bruno's Art of Memory

## 是什么
将宇宙本身建成一座记忆宫殿——用黄道三十六旬星的魔像填满灵魂，最终以"英雄的狂热"点燃认知之火。

## 可借元素
**性相映射：** 灯（以内在光辉照见被遗忘的知识）、启（将宇宙的结构书写进灵魂的褶皱）、铸（以记忆为熔炉，锻造新的认知形态）。布鲁诺的记忆术不是记忆技巧，而是通过重组灵魂中的意象来改变灵魂本身——这与密教信徒通过积累秘传知识来改变自身性相的过程如出一辙。

**适用实体类型：** 仪式 / 影响 / 典籍 / 追随者。

**具体用法：**
1. **仪式卡**——"封印之封印"（Seal of Seals）：选择一个追随者，将其暴露在你拥有的所有典籍之下。追随者获得一个永久性的启之性相提升，但每次使用此仪式都冒着"思维燃烧"的风险——追随者可能直接进入狂乱状态（转变为不受控的"狂热者"），概率与你拥有的典籍数量成正比。
2. **影响卡**——"英雄的狂热"：获得后，显著提高所有需要启之性相的仪式的成功率，但持有期间每季都可能随机触发一个灵感——其中一些是真的，一些是危险的谎言。三季后消散，消散时留下"灰烬之思"。
3. **典籍**——"论阴影之理念"（De Umbris Idearum）：阅读后解锁"记忆剧场"——一种新的行动类型：你可以将一张卡的属性"铭刻"进另一张卡中，进行性相混合。
4. **追随者**——"记忆术士"：特有能力——当场上有"猎手"试图调查你时，记忆术士可以用他灵魂中的宇宙图像"覆盖"猎手的记忆，使其跳过证据收集阶段一次。每次使用消耗他的一个记忆槽位，三次后他成为一个记忆全失的"空心人"。

**关键意象：** 轮转的三十六旬星之轮、记忆场所如星辰般排列在灵魂的穹顶、火焰中站着微笑的男人、玫瑰与百合交织的记忆锁链。

## 来源
焦尔达诺·布鲁诺（Giordano Bruno, 1548-1600），多明我会修士、宇宙论者、赫尔墨斯主义者。主要著作包括《论阴影之理念》（1582）、《论原因、本原与太一》（1584）、《论英雄的狂热》（1585）。1600年因坚持日心说与赫尔墨斯宇宙论被宗教裁判所烧死在罗马鲜花广场。
