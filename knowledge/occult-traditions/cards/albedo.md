# 白化 / Albedo

## 是什么
炼金的第二阶段——净化、洗涤、白石的显现。黑暗褪去，月光照亮清洁之物。

## 可借元素
**性相映射：** 心（净化、整全、健康）、灯（从黑暗中浮现的清明之光）。白化是心相与灯相的交汇——不只是变"亮"，更是变"干净"。在CS体系中，没有一种单一的性相可以表示"纯净"，所以白化天然是混合性相的仪式。

**适用实体类型：** 仪式 / 治疗机制 / 状态净化。

**具体用法：**
1. **净化仪式**——"白化"：消耗一份"黑化"后的原料（带有"可塑"标签的冬+蛾混合原料）＋一份心相影响（如"安宁"、"脉动之歌"）。产出：该原料的"净化"版本——移除所有冬相和蛾相成分，只保留原初性相，并附加一点灯相（"澄明"标记）。净化后的原料可用于后续的赤化仪式，或直接作为高级祭品使用。
2. **状态净化器**——"白化之浴"：玩家可以在任何季节执行此行动。消耗一份心相原料（如"不朽之血"）来移除自己身上的一项负面状态或恐惧卡。仪式成功的概率与你的心相等级成正比。如果用于追随者，可以移除追随者的创伤标记——但每移除一道创伤，追随者永久损失一点性相（创伤已内化为他们的一部分）。
3. **白化追随者**——"白化者"称号：只有经历过黑化并苏醒的追随者才能获得此称号。白化者在每次成功执行心相或灯相任务后，有概率将该任务的产出提升一阶（"纯净之手"）。但他们无法执行任何需要杯相或刃相的任务——"纯净"排斥欲望与暴力。
4. **意象集**——白天鹅、银月、白色的女王（Regina Alba）、月光洗涤下的白色石头、银色的哲人汞。

**仪式配方示例：** 白化仪式 —— 消耗：任意"可塑"原料＋一份心相影响＋一份灯相工具（如"明镜"）。产出：原料的"净化"版（原性相保留，冬/蛾相移除，附加灯相标记）。

## 来源
中世纪炼金术的白化阶段（albedo, leukosis）。在阿拉伯炼金术中被称为"بَيَاض"（bayad/白）。常见于《升起的黎明》（Aurora Consurgens）和《哲学家的玫瑰园》。荣格心理学将其诠释为"阿尼玛的整合"——从阴影中走出后，面对灵魂中的阴性/内在面。在炼金操作中，白化标志着哲人汞完成了对不纯物的洗涤。
