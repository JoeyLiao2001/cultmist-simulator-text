# 迪伊的天使魔法 / John Dee's Enochian Magic

## 是什么
透过黑曜石的镜面，字母被天使逐一呼出——一套只对世界末日有效的语言。

## 可借元素
**性相映射：** 灯（天使的光辉透过石镜折射为话语）、启（被逐字念出的语言——语言作为开启实在之门的钥匙）、秘史（以诺语据说是亚当堕落前使用的语言——人类历史原点的秘密）。迪伊系统的独特之处在于：它不是一套靠祈请或献祭驱动的魔法，而是一套纯粹由语言构成的引擎——四十八声呼唤、四大守望塔、三百六十天使之名。在这个系统中，发音本身就是力量的释放。

**适用实体类型：** 仪式 / 工具 / 影响 / 典籍 / 总部升级。

**具体用法：**
1. **仪式卡**——"四十八声呼唤"（48 Calls/Keys）：这实际上不是一个仪式，而是四十八个仪式——每个呼唤对应解锁一种漫宿领域（对应迪伊的三十重"爱忒" / Aethyrs）。需要消耗相应的天使之名（一份典籍提供的抄本），才能打开对应领域的传送门。每开一重门，世界的时间流逝就慢一分——三十重门全开时停的那一天，旧世界终结。这可以直接与密教的"渴望日"（Desire）机制挂钩——打开足够多的门，就是通往终局的路径。
2. **工具卡**——"示石"（Shewstone）：一块椭圆形的黑曜石，置于"真理封印"（Sigillum Dei Aemeth）之上。持有后，所有需要灯之性相的仪式成功率翻倍，但每次使用后需要一名具有"预言"特质的追随者去"凝视"（消耗一季），否则石头冷却为普通黑曜石。
3. **影响卡**——"天使音"：一种罕见的声音类影响——当持有示石且场上有特定相位时，有概率获得天使之声作为影响。使用它可以在一次仪式中直接召唤一位天使具名者，无需消耗任何其他召唤原料。但这张影响极其短暂（一季），且持有期间你的总部所有行动都会因为被"注视"而降低成功率。
4. **总部升级**——"大地的伟大书板"（Great Table of Earth）：允许你将总部的一部分改造成以四大守望塔（东风水火土）为模板的仪式场。每建成一塔，对应性相的追随者在总部中自动获得一级能力提陞。但四塔齐备之日，世界会开始"注意"你——执法机构的兴趣增长速度变为两倍。
5. **典籍**——"以诺之书"：阅读后获知天使们的名字列表。单独看只是符号，但和其他典籍搭配时，可以作为"翻译字典"——用天使之名替换你已有仪式中消耗的任意语言类原料。

**关键意象：** 黑曜石在烛火中映出的不是房间的倒影而是另一片星空、天使的字母不是被写下而是被"听见后铭刻"在蜡版上、红宝石般的激光在字母排列中闪烁、盲眼灵视者凯利逐字诵读而迪伊逐笔记下。

## 来源
约翰·迪伊（John Dee, 1527-1608/09），伊丽莎白一世女王的首席占星顾问、数学家、炼金术士、赫尔墨斯主义者。从1582年至1589年间，迪伊与灵视者爱德华·凯利（Edward Kelley）进行了一系列"天使对话"（Angelical Conversations），期间凯利声称通过示石看到了天使，天使们逐字母教会了他们一套完整的语言——以诺语（Enochian）。主要文献为迪伊的手稿《以诺之书》（Liber Logaeth），后经金色黎明秘教会的梅瑟斯（S.L. MacGregor Mathers）整理和推广。示石现存于大英博物馆。克劳利曾尝试重现天使对话，声称见到了自己的守護天使艾华斯。
