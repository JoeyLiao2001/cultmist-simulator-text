# 赤化 / Rubedo

## 是什么
炼金的最终阶段——完成、哲人石之达成。红国王加冕，凤凰从灰烬中升起，一切转化到此终结。

## 可借元素
**性相映射：** 铸（终极的转化与成型）、杯（圆满、成就、不朽之宴）。赤化不只是"完成"，而是"再也不能被逆转的完成"——这是铸相区别于蛾相的关键：蛾是永不确定的过程，铸是永不回退的结果。

**适用实体类型：** 终极仪式 / 胜利条件 / 最高阶工具。

**具体用法：**
1. **哲人石工具**——"赤化之石"：游戏中最高阶的工具卡，但不可直接制作。只有当你完成完整的炼金三阶段（黑化→白化→赤化）之后，才能通过"赤化仪式"获得。哲人石可以在任何仪式中充当任意性相的最高阶工具——但每使用一次，需要消耗一份杯相原料来"喂养"它（否则它将退化一阶——不是碎裂，而是退回白化阶段）。
2. **胜利条件**——"赤化结局"：如果你完成了赤化仪式并且持有"赤化之石"，你可以选择一项标准的胜利条件（如灯相飞升、杯相飞升等）——并在其基础上升级为"赤化版本"的结局。赤化版本不改变结局的性质，但改变结局的质量：你的飞升不再是加入已有司辰，而是以你自身成为新的移涌——因为你的存在已经被哲人石重铸为不可逆的神性。
3. **终极仪式**——"赤化仪式"：前提你必须持有一份经历过黑化和白化的"净化"原料，还需要：一份杯相最高阶原料（"圆满"）、一份铸相最高阶工具（"熔炉碎片"或更高）、以及一朵"凤凰之羽"（稀有掉落——仅在某个追随者达到最高级且在濒死状态下存活时获得）。赤化仪式成功后，净化原料转化为哲人石。
4. **意象集**——赤红的国王（Rex Rubus）、从火焰中飞升的凤凰、红色的哲人石、红白玫瑰的婚礼（与炼金婚礼合并意象）、黄金（太阳的金属）。

## 来源
中世纪炼金术的赤化阶段（rubedo, iosis）。也可称为"泛红"或"红化"。在《哲学家的玫瑰园》中，赤化标志着"神圣的国王"的诞生。荣格心理学将其诠释为"自性化"（individuation）的完成——意识与无意识的最终整合。在炼金操作层面，赤化意味着哲人石的雏形已经形成，现在需要在最高温度中"定色"——石头从此不朽且不可逆。
