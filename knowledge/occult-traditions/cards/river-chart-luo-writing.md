# 河图洛书

**是什么** 上古圣人伏羲见龙马负图出于黄河，大禹见神龟负书出于洛水。图与书皆以黑白点阵构成宇宙数理之秘，八卦、五行、九宫皆由此出。

**可借元素**
- **密教准则**：秘史（已失落的起源知识）与启（天启、洞开）。图即是密码，解读即等于窥见世界底层结构。
- **适合实体类型**：知识 / 藏宝图 / 解密物。可作为"终极学问"——获取后解锁特殊的合成配方，允许玩家从不相关的准则中提取出高阶秘传。
- **命名建议**：河图 → "龙马阵图""数源经""十数天衍"。洛书 → "龟背九宫""洪范九畴""三阶幻方"。图与书分开收集，合一时触发揭示。
- **具体意象**：白点为阳数（奇数），黑点为阴数（偶数）；图中五与十居中，书戴九履一；排列看似简单，反复推演可生出无限变化。

**来源** 《尚书·洪范》《易传·系辞》，后世纳入道家与术数体系。
