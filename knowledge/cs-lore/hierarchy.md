# 密教模拟器 实体位阶体系 (Entity Hierarchy of the Cultist Simulator Universe)

> **注**: 本文档基于 `cs_dataset.jsonl` 中的实际游戏文本片段构建。证据以 `> 引用` 形式呈现，标注来源条目 ID 与分类。

---

## 位阶总览（自下而上）

```
司辰 (Hour)         ── 隐秘诸神，统治漫宿与醒时世界法则
   ↑
具名者 (Name)       ── 司辰的直属仆从/化身，半神级存在
   ↑
长生者 (Long)       ── 通过三尖门的不死者，已摆脱凡人寿命
   ↑
门徒 (Exalted)      ── 被擢升至最高阶的教团成员，已将一项原则磨砺至极致
   ↑
信徒 (Believer)     ── 已发下正式誓言的教团追随者，初窥奥秘
   ↑
熟人 (Acquaintance) ── 对无形之术有兴趣的凡人，尚未正式加入教团
   ↑
凡人 (Mortal)       ── 未经启蒙的普通人类
```

## 1. 凡人 (Mortal)

### 定义

未经启蒙、不知晓无形之术存在的普通人类。众生皆以凡人之躯降世，在肉体的限制中度过一生。凡人可以被杀死、被逼疯，也可以被引入秘教世界。

> **证据**: `"我们如此习惯活着，以至于不愿意死亡。"——托马斯·布朗`（元素 `mortal`）

> **证据**: `"凡人将变作死尸"`（元素 `killmortal`）

> **证据**: `"凡人将沦为疯子"`（元素 `derangemortal`）

### 关联性相

无特定性相。凡人是未开化的状态，不携带任何性原则的印记。但他们的基础能力（健康、激情、理性）分别对应：

- **健康 (Health)** → **心**（`["心", "原料", "能力"]`）
- **激情 (Passion)** → **蛾**（`["蛾", "原料", "能力"]`）
- **理性 (Reason)** → **灯**（`["灯", "原料", "能力"]`）

> **证据**: `"属于我的这具身体。世间存在各种各样的身体，但这一具是属于我的。我的头脑需要它，正如菌菇需要泥土。"`（元素 `health`）

### 如何晋升至熟人

凡人通过接触到无形之术的暗示、结识同道中人，或对世界表皮之下的形状产生兴趣，从而成为"熟人"。

> **证据**: `"我或许能找到同道中人。"`（元素 `dayacquaintances`，时节类）

> **证据**: `"多萝西是个热心的人，有着涉猎广泛的读书习惯。她为我所暗示的世界表皮下的形状而着迷。"`（元素 `dorothy_a`，熟人阶段）

### 凡人盟友示例

| 名称 | 性相 | 描述 |
|------|------|------|
| 理发师祖尔菲亚 | 蛾 | 剪刀从不离身 |
| 深掘者欧索丽娜 | 铸 | 曾为狮子匠服务 |
| "魅裔"瓦西尔 | 杯 | 血统中非人类的部分并不明显 |
| 面包师多明卡斯 | 冬 | 脸上随时带着微笑 |
| 女猎手拉拉·柴玛 | 刃 | 身躯灵巧、面容严肃 |
| 走私商人乔吉奥 | 启 | 神情忧郁、带哲学家气质 |
| 灵视者米蕾娅 | 灯 | 神情如在梦中一般恍惚 |
| 活体解剖家雅宁斯 | 心 | 拆解和装配方面的专家 |

### 力量与局限

- **力量**: 可被雇佣从事各种任务；每人都带有微弱的天赋偏向
- **局限**: 不能参与仪式；不知晓密传；使用后会疲劳（出现"枯竭"状态）

> **证据**: 凡人盟友均有对应的 `fatigued` 变体，如 `ally.mortal.fatigued` 带有 `["凡人", "枯竭", "盟友", "伙伴"]`

## 2. 熟人 (Acquaintance)

### 定义

对无形之术产生兴趣、但尚未正式加入教团的凡人。他们知晓世界并非表面上那样简单，但仍站在门槛之外。用游戏的术语说：**"非友，亦非敌"**。

> **证据**: `"非友，亦非敌"`（元素 `acquaintance`）

### 关联性相

熟人阶段与具体性原则的关联尚不明显，但他们已有各自的成长方向（对应八个主要性相之一）：

- 奥克莱尔 → 冬
- 卡特·卡罗 → 灯
- 思雷 → 灯
- 芮妮拉/萨利巴 → 杯
- 伊妮德/内维尔 → 启
- 萝丝/维克多/埃尔里奇 → 刃
- 莱德劳/崔斯坦/万查妮 → 铸
- 克罗薇特/多萝西/里奥 → 心
- 西尔维娅/伊莎贝特/波特 → 蛾

> **证据**: 以 `dorothy_a` 为例 —— `["凡人", "熟人"]`，无性相标记；升为 `dorothy_b` 后获得 `["凡人", "追随者", "心"]`

### 如何晋升至信徒

玩家需要向熟人揭示教团的奥秘和誓言。工具性元素 `promotiontodisciple` 存在但仅描述为：**"该让他发下更进一步的誓言了"**。晋升需要以相应的密传进行说服。

> **证据**: 每个可招募的熟人角色都有 `_a`（熟人）、`_b`（信徒）、`_c`（门徒）三条完整的进阶链。

### 典型熟人示例

| 姓名 | 描述 |
|------|------|
| 奥克莱尔 | 有着学者的面孔和病态的气质 |
| 卡特·卡罗 | 剧场灵媒兼算命师，可能还有更深层次的天赋 |
| "教皇"克利夫顿 | 散发着神秘主义者气息的非正统天主教徒 |
| 克罗薇特 | 快乐是那种容不下不快乐的快乐 |
| 伊莎贝特 | 总在月光下感到最自在，拥有绝佳的夜间视力 |
| 思雷 | 天文学者兼先锋派诗人，因健康不济对怪力乱神产生兴趣 |
| 维奥莱特 | 不遗余力贡献时间，去缓解垂死之人的痛苦 |
| 萨利巴 | 有着惊人胃口，早就过了通过满足欲望获得欢愉的阶段 |
| 西尔维娅 | 九岁在树林里失去一只眼睛，每年圣烛节回到埋葬眼睛的地方 |
| 芮妮拉 | 在三个不相邻的辖区被判三次无罪释放 |

## 3. 信徒 (Believer / Disciple)

### 定义

**已发下誓言、正式加入教团的追随者。** 他们初窥无形之术的奥秘，将自己的才能投入一个特定性相原则的修炼之中。在游戏术语中，"信徒"和"门徒"都以 `follower`（追随者）标注，但"信徒"等级（`_b`）尚未达到门徒（`_c`）的高度。普通追随者（未指定原则）的描述为：**"你的学生、合伙人、护卫、拥护者、被你欺骗、被你当作祭品的……伴侣。"**

> **证据**: `"你的学生、合伙人、护卫、拥护者、被你欺骗、被你当作祭品的……伴侣。"`（追随者 `follower`）

### 关联性相

每个信徒获得其所追随的原则性相：

| 信徒 | 性相 |
|------|------|
| 奥克莱尔 | 冬 |
| 卡特·卡罗 / 克利夫顿 / 思雷 | 灯 |
| 克罗薇特 / 多萝西 / 里奥 | 心 |
| 莱德劳 / 道格拉斯 / 崔斯坦 / 万查妮 | 铸 |
| 西尔维娅 / 伊莎贝特 / 波特 | 蛾 |
| 伊妮德 / 内维尔 | 启 |
| 萝丝 / 维克多 / 埃尔里奇 | 刃 |
| 芮妮拉 / 萨利巴 | 杯 |

> **证据**: `"奥克莱尔沉静，做事细致而又冷酷。"`（`auclair_b`，`["凡人", "追随者", "冬"]`）

> **证据**: `"萝丝是个狡黠的姑娘，但她忠诚。"`（`rose_b`，`["凡人", "追随者", "刃"]`）

### 如何晋升至门徒

需要进一步教授高阶密传、授予教团内的更高地位。晋升条件由 `promotiontodisciple` 元素表示：**"该让他发下更进一步的誓言了"**。实际上，通过向其传授对应原则的高等级密传可将其擢升为门徒（Exalted）。

| 奥克莱尔 | 学者面孔 → 沉静、细致、冷酷 |

| 伊莎贝特 | 月光下自在 → 睡眠断断续续，黄昏醒来 |
| 芮妮拉 | 不会被判罪 → 在追寻食欲时保持谨慎 |
| 伊妮德 | 目光不聚焦 → 能看见他人不视之物 |
| 萨利巴 | 惊人胃口 → 了解鲜血，尚未了解生育 |
| 西尔维娅 | 埋眼睛 → 在屋脊、枯河、威士忌、变魔术上有很深造诣 |

## 4. 门徒 (Exalted / Key Disciple)

### 定义

**"初窥奥秘"** —— 已将某项原则磨砺至极致，获得"天赋"（Gift）标记的追随者。这是凡人追随者所能达到的最高境地。在此之后，他们"无法再进一步晋升……至少在我麾下不能"。

> **证据**: `"初窥奥秘。"`（元素 `disciple`）

> **证据**: `"此人无法再进一步晋升……至少在我麾下不能。"`（元素 `exalted`）

### 关联性相

每个门徒带有其主原则的性相，附加**"门徒"**和**"天赋"**标签。部分门徒还获得一个辅助性相。

> **证据**: `"奥克莱尔不怎么说话。她对自己的言语斟词酌句。她连动作也是能简则简，生怕浪费气力。还有，人们从没见她眨过眼。"`（`auclair_c`，`["凡人", "追随者", "门徒", "天赋", "冬"]`）

### 力量与局限

- 门徒是所有凡人追随者中最强大的
- 带有"天赋"标记，在对应原则的挑战中拥有最高成功率
- 某些特殊门徒拥有抗病、教诲语言等能力
- 无法再在此系统中进一步晋升 —— **"至少在我麾下不能"**

### 门徒的终极命运

如果玩家以特定诱惑飞升至长生者，且选择了对应的使徒（Apostle）遗产，则门徒级别的追随者可能会继续追随玩家。游戏中也有**"对手飞升"**系统——如果玩家不接受特定欲望，某个门徒将自行尝试飞升，成为玩家的对手。

## 5. 长生者 (Long)

### 定义

**已通过三尖门（Tricuspid Gate）的不死者。** 长生者是已摆脱凡人寿命的存在，但尚未成为具名者。他们或是为司辰效力，或是仇恨司辰，或是逃避于午港（Port Noon）。杀死长生者是极其困难的。

> **证据**: `"杀死一个已经通过三尖门的人是非常困难的。而且这里有很多这样的人。不过也许我们可以欺骗一个，或者投其所好。"`（元素 `guardian_long`）

> **证据**: `"每个不死者都要面对下列抉择：侍奉一位司辰，打回凡籍，还是自行消灭。这座阳光明媚，名为'午'的小小岛港是拒绝上述选项者的流放地。"`（地点 `vaulteveningislesdefault`，午港）

### 关联性相

长生者的性相取决于他们的来源和所侍奉的司辰。数据集中的长生者分为：

1. **形成途中的长生者** (`_r_d` 后缀) —— 玩家门徒转化而来的对手
2. **永生不死的敌人** —— 特定的具名长生者对手（如 J.C.）

### 如何成为长生者

玩家通过以下三种飞升路径之一晋升：

| 路径 | 司辰 | 印记 | 中文名 |
|------|------|------|--------|
| 启明 (Enlightenment) | 守夜人 | 光之印记 | 灯之长生者 |
| 力量 (Power) | 白日铸炉 | 塑之印记 | 铸之长生者 |
| 感官 (Sensation) | 赤杯 | 悦之印记 | 杯之长生者 |

> **证据**: `"我可以向名为守夜人的司辰寻求永生与启明。他的触碰能为我带来独特的食欲，以及印记。"`（元素 `enlightenmentmarks`）

> **证据**: `"我可以向名为白日铸炉的司辰寻求永生与力量。她的触碰能为我带来独特的食欲，以及印记。"`（元素 `powermarks`）

> **证据**: `"我可以向名为赤杯的司辰寻求永生与愉悦。她的触碰能为我带来独特的食欲，以及印记。"`（元素 `sensationmarks`）

#### 飞升阶段（印记系统）

每个飞升路径经历七个阶段的蜕变。以"启明"为例：

1. **诱惑** —— "我突然意识到我手里握着前往智慧的钥匙"
2. **追奉** —— "我已奉身于追求更高层次的理解"
3. **第三印记** —— "守夜人，即瞳中扉，触碰了我。现在每天我的好奇心都会增强：这是一种急迫的，生理上的需求，就像饥饿一样折磨我"
4. **第四印记** —— "眼中亮着的光……大多数情况下，人们只有在油灯熄灭时或者午夜的黑暗之中才能看到它……我开始日渐消瘦了"
5. **第五印记** —— "光芒会从皮肤乌青或被剥皮之处泄露出来……我现在相当瘦弱。我总是很饥饿"
6. **第六印记** —— "虎口间的皮肤可以拉到小指关节处……我的皮肤已经变得非常非常松弛了……我的双眼丧失了属于人类的色泽"
7. **最终** —— 穿过三尖门 / 王皮之门（不同路径不同门关）

> **证据**: 参见 `ascensionenlightenmenta` 至 `ascensionenlightenmentf` 系列元素

#### "力量"路径的印记变化

> **证据**: `"铸炉在我肉体上点燃了回应的火花。我极度渴求火焰的热力。"`（`ascensionpowerc`，第三印记）

> **证据**: `"虽然火焰让我遍体伤疤，但想炙伤我的肉体也是越来越难。我现在开始渐渐习惯火焰，如同一条龙或一块石头。"`（`ascensionpowerd`，第四印记）

> **证据**: `"第五印记：我的身体没有一根毛发。我变得像魔鬼一样强壮。如果我离开了火焰，我就会不停颤抖。"`（`ascensionpowere`）

> **证据**: `"我满身伤疤，四肢都很难弯屈。我不愿将我的面容暴露在众人的目光下。我能手握炙热的煤炭而毫无知觉。"`（`ascensionpowerf`，第六印记）

#### "感官"路径的印记变化

> **证据**: `"赤杯给予我温暖。它的干渴让我的嘴变干，手发痒。我知道这是第三印记。"`（`ascensionsensationc`）

### 长生者的力量与局限

**力量**:

- 不死之身，难以被杀死
- 可在城市中建立关系网（`long.connections`）和积累力量（`long.might`）
- 可通过梦境（漫宿）攻击对手（`long.dreamwork`）
- 可打造梦境侵扰敌人（`poison.dread.longlesson`）

> **证据**: `"随着时间推移，这名长生者将在城市中建立起关系网，增强他使用'阴谋'陷害我们的能力。"`（`long.connections`）

**局限**:

- 仍然处于具名者和司辰的压制之下
- 晋升具名者会遭到其他长生者的强烈反对
- 某些特殊武器可以伤害甚至杀死他们

> **证据**: `"长生者有可能死去吗？让我们来试试。"`（元素 `long_wound`）

## 6. 具名者 (Name)

### 定义

**司辰的直属仆从或化身。** 具名者是半神级存在——比长生者更强大，但仍是司辰的从属。他们是司辰在世间和漫宿中的执行者。每个具名者都与其侍奉的司辰共享性相。

> **证据**: `"上校在不同的时代有着不同的形象。他目盲；他耳聋；他不受创伤；他不容违抗。他的具名者皆如极年迈之人一般狡诈。"`（碎片 `fragmentedged.colonel`）

> **证据**: `"狮子匠，按司辰的标准来看尚显青涩，而按战争的标准来看相当老成。他的具名者皆是刚被招募的新成员，其吟诵有一种野蛮的力量。"`（碎片 `fragmentedged.lionsmith` 和 `fragmentedgee`）

### 关联性相

具名者的性相反映其侍奉的司辰：

| 具名者群体 | 司辰 | 性相 |
|-----------|------|------|
| 上校的具名者 | 上校 | 刃、灯、冬 |
| 狮子匠的具名者 | 狮子匠 | 刃、铸、心 |
| 裂分之狼的具名者 | 裂分之狼 | 刃 |
| 铸之具名者 | 白日铸炉 | 铸 |
| 杯之具名者 | 赤杯 | 杯 |
| 骄阳的具名者 | 骄阳 | （已崩解，碎片尚存）|

> **证据**: `"'裂分之狼具三性：毁灭性，毁灭性，以及，毁灭性。' 学习司辰的技艺不代表你必须喜爱它。"`（碎片 `fragmentedged.exile`）

> **证据**: `"铸之具名者都念诵过这些话语。"`（碎片 `fragmentforged`）

> **证据**: `"杯之具名者的典仪进程有刺耳尖叫声的参与。我们要尽可能地重现这些声音，以求得其所提供的搏动之力。"`（碎片 `fragmentgraild`）

### 如何成为具名者

成为具名者需要：
1. 首先成为长生者（通过三尖门）
2. 被一位司辰选中和擢升
3. 长生者之间的竞争极为激烈——因为晋升意味着接近司辰之位

> **证据**: `"许多长生者为司辰效力。他们强烈反对晋升新具名者，只是因为他们将来可能，仅仅是可能，篡夺司辰之位。"`（`long.enemy`）

### 具名者的崩解

当一位司辰被毁灭（如骄阳被分裂），其具名者也会随之崩解。

> **证据**: `"当骄阳被分裂，它的具名者亦随之崩解。这些是曾为整体的那些具名者们的碎片。"`（碎片 `fragmentwinterf`）

### 具名者相关的技艺

高等级的碎片密传直接以具名者的名义命名：
- `fragmentedgeg` —— 武力之秘 —— "上校和狮子匠是肉源之神，但是裂分之狼是血源之神。"
- `fragmentforged` —— 巧匠祷文 —— "铸之具名者都念诵过这些话语"
- `fragmentgraild` —— 酩酊祷文 —— 杯之具名者的典仪
- `fragmentlanternf` —— 攀升梵咒 —— 刀刃阶梯的考验

### 无名的具名者

数据集多处提到一位特定的"无名的具名者"（Nameless Name），与丝毧（Moth-Hour）有关联。

> **证据**: `"据说无名的具名者在醒时寻找秘密，并把它们作为礼物带给林地的丝毧。同样，猫也会把老鼠带到家里作为礼物送给主人，但我不认为这总能得到相同程度的认可。"`（元素 `pentiment.display`）

> **证据**: `"丝毧无名的具名者经行此地"`（阿姆斯特丹居住地）; `"也许无名的具名者会用她的迷雾遮蔽我"`（伦敦居住地）; `"如果丝毧那位无名的具名者也会在白日出行，那她想必会在此地出没"`（威尼斯居住地）

### 具名者与神器

某些神器本身被认为是具名者。例如，比德之刃（Biedde's Blade）被大敌列为上校的具名者：

> **证据**: `"一说'比德'是十八世纪图卢兹一名喜欢收集刀具的外科医生兼拦路劫匪。另一说'比德'只是阿拉伯语'蚊子'的讹传。但我的大敌曾经把比德列为上校的具名者。"`（工具 `weapon.d`，比德之刃）

### 持盾蛇卫 —— 猎杀长生者之人

**"持盾蛇卫"**（Snake-Guard）是从上校处习得技艺、专门猎杀长生者的战士。他们可能本身就是一种特殊的具名者。

> **证据**: `"图拉古拉里，即受雇猎杀长生者之人，曾从上校麾下的持盾蛇卫那里习得了他们的技艺。这把投枪的尖头是一颗剑齿兽牙。"`（工具 `weapon.f.colonel`）

## 7. 司辰 (Hour)

### 定义

**隐秘诸神，统治漫宿与醒时世界的至高存在。** 司辰共有三十位（一说三十六位，但已有部分失落）。他们并非全知全能，各自掌管特定的原则领域，彼此间存在复杂的敌对、联盟与从属关系。司辰各有其"颜色"。

> **证据**: `"司辰各有其颜色，但颜色只存在于有光的地方。"`（碎片 `fragmentlantern`）

> **证据**: `"六位已逝。五位源自岩石，一位源自光。现都归于虚界。"`（碎片 `fragmentwintere`）

> **证据**: `"六者已失，卅者留存。"`（工具 `painting.palest`，淡白至极的画作）

### 司辰的起源分类

根据《司辰的起源》和《缅怀诸神》等文献：

1. **石源之神（Gods-from-Stone）** —— 先于人类存在的远古司辰，大部分已被灭绝
2. **光源之神（Gods-from-Light）** —— 源自辉光（Glory）的司辰
3. **肉源之神（Gods-from-Flesh）** —— 曾为凡人、后擢升的司辰
4. **血源之神（Gods-from-Blood）** —— 最特殊的起源，如赤杯（第一位血源之神）和裂分之狼

> **证据**: `"作者一个无名的高卢人学徒……此人带着遗憾之情讲述了先于人类的司辰——石源诸神的毁灭。"`（文献 `memoryofgods`）

> **证据**: `"上校和狮子匠是肉源之神，但是裂分之狼是血源之神。狡诈、力气与痛苦乃胜利的三大根基……"`（碎片 `fragmentedgeg`）

> **证据**: `"白日铸炉、弧月和昕旦是光源之神。汞、银和金乃功业的种穰……"`（碎片 `fragmentforgeg`）

> **证据**: `"圣杯是第一位血源之神。制花人来自于光。拾滩鸦曾是肉身。"`（碎片 `fragmentgrailg`）

> **证据**: `"弧月源于光，生于月亮。"`（碎片 `fragmentknockg`）

### 司辰之力

- 统领漫宿（Mansus），即诸梦之居屋
- 各自掌管特定的原则性相
- 能授予凡人以印记，开启飞升之路
- 能擢升长生者为具名者
- 互相制衡——在漫宿中抗衡，消解不可能之事，编织历史

> **证据**: `"在漫宿中，司辰互相抗衡。纷争平定后，祂们消解了不可能之事，歌颂了可能之事，将纠缠交错的已成之事细密编织成如金色丝带般的未来。"`（碎片 `fragmentsecrethistoriese`）

### 司辰的死亡与继承

司辰并非永恒——已有司辰被毁灭：

- 六位已逝（五位石源 + 一位光源）
- 骄阳被分裂——"置闰"事件
- 七蟠被斩灭——"七步斩七蟠"
- 飞蛾王在轰雷王的大腿处孵化（轰雷王之死与飞蛾之生）

司辰之间存在继承关系：当一位司辰死亡，另一位可能从其残骸中诞生或继承其领域。

## 特殊类别

### 召唤物 (Summons / Spirits)

**"此人本不属于这个世界。"**

召唤物是通过仪式从漫宿或其他界域召唤而来的非人实体。它们不属于正常的位阶体系，但拥有强大的力量。每个召唤物都绑定特定的性相组合。

> **证据**: `"此人本不属于这个世界。"`（召唤物 `summoned`）

> **证据**: `"召唤物变作调味影响或阻碍召唤之人。"`（`killsummoned`）

#### 主要召唤物列表

| 召唤物 | 性相 | 类型 | 描述 |
|--------|------|------|------|
| **坩埚王** (King Crucible) | 铸、刃 | 吞噬者 | "我于荒漠中等待、于废墟中燃烧——金铁为水——土石化蜡——肉身成烟——来入我内，不复续存" |
| **第二干渴者伊泽姆** (Ezeem) | 杯、刃 | 吞噬者 | "改变我。改变我。用力改变我啊。现在——轮到我了" |
| **特蕾莎** (Teresa) | 灯、秘史 | 教师 | "噢，世界是平的；噢，可它是平的！我不会久留，我不该久留。告诉我个秘密" |
| **原生先知** (Raw Prophet) | 杯、蛾 | 欺骗者 | 时而分散，时而聚集；时而预言灾难，时而宣扬食欲 |
| **擂击者** (Percussigant) | 心、刃 | 吞噬者 | 热爱叉铃震颤与定音鼓的敲打；行动展现对《被剥皮的马西亚斯》的喜爱 |
| **迹形** (Hint) | 灯、刃 | 吞噬者 | 躲在舞台后面、镜子后面、你后面；从门底滑入，用玻璃杀戮，却活不过活人的注视 |
| **汽灵** (Caligine) | 铸、蛾 | 欺骗者 | 在漫宿边界，铸炉四隅——可侵染敌人梦境的一团狡猾蒸汽 |
| **镜中少女** (Maid-in-the-Mirror) | 冬、刃 | 吞噬者 | "路为玻璃，墙为刀刃，步步见血"——穿过孔雀之门侍奉残阳的亡者 |
| **无声亡者** (Voiceless Dead) | 冬、蛾 | 欺骗者 | 通过纯白之门进入漫宿，将说话能力永远留在门槛外；落灰的皮囊会遵从命令 |
| **破碎行尸** (Shattered Risen) | 冬、刃 | 残骸 | 撕裂之力拖回半死不活的状态，手掌是骨片交织成的利爪 |
| **抽芽行尸** (Burgeoning Risen) | 冬、蛾 | 残骸 | 林地植物覆满器官，缠住心脏，颅骨中胀大如匏瓜；骨头是腐木 |

#### 召唤物的特殊能力

- **吞噬者 (Devourer)**: 可吞噬激情（失物之灯）或理性（褪思者）以维持自身
- **欺骗者 (Deceiver)**: 可执行隐蔽任务
- **教师 (Tutor)**: 可教授稀有语言（如特蕾莎教富奇诺语，坩埚王教曼达安语，伊泽姆教弗里吉亚语）
- **抗病大师**: 可提供疾病防护
- **残骸 (Remnant)**: 从死亡中苏醒的不完整存在

> **证据**: `"我可以与此物交谈，向它献上我的激情。"`（`spiriteatspassion`，失物之灯）

> **证据**: `"我可以与此物交谈，向它献上我的理性。"`（`spiriteatsreason`，褪思者）

> **证据**: `"我可以与此物谈论我的健康，它能给予人针对疾病的防护。"`（`spiritsicknessprotector`，抗病大师）

#### 召唤物的局限

- 有召唤时限，可"重燃"（延长停留）
- 被杀死后会留下残骸或影响
- 某些召唤物被击败后可能震怒，不能再被召唤

> **证据**: `"我进食了那只坩埚留在世间的印痕。往最轻了说，这会冒犯到他。他可能会震怒。我不应该再召唤他了。"`（`spirit_forgee_edge.consumed`）

### 猎物 / 敌对 (Rivals / Enemies / Apostles)

这类角色代表了玩家在飞升之路上的竞争者，分为两类：

#### a) 长生的对手 (Long Rivals)

门徒级追随者若被"放手"，会自行尝试飞升，成为"形成途中的长生者"——竞争同一个飞升位置的对手。他们会建立人脉、积累力量、在梦中学知。

> **证据**: `"许多长生者为司辰效力。他们强烈反对晋升新具名者，只是因为他们将来可能，仅仅是可能，篡夺司辰之位。"`（`long.enemy`）

> **证据**: `"随着时间推移，这名长生者将在城市中建立起关系网，增强他使用'阴谋'陷害我们的能力。"`（`long.connections`）

> **证据**: `"虽说对手飞升到日之居所的可能性微乎其微。但如果他成功了，我或许会失去飞升的机会。"`（元素 `rivalmarks`）

对手的行动类型：
- **袭击 (Assault)** —— 直接攻击
- **阴谋 (Intrigue)** —— 诡计陷害
- **梦境 (Dreams)** —— 通过漫宿攻击
- **缓期执行 (Downtime)** —— 休整增强

> **证据**: 参见 `strategy.assault.revealed`, `strategy.intrigue.revealed`, `strategy.dreams.revealed` 系列

对手会"学有所成"（`long.lesson`）：**"我们的敌手在观察和学习"**。理解后会在特定方面增强（人脉扩张 `long.lesson.connections`、力量增强 `long.lesson.might`、习学灵魂 `long.lesson.dreamwork`）。

#### b) 使徒 (Apostles)

使徒是玩家（已成为长生者）的追随者。在DLC"使徒"（Apostle）遗产中，玩家扮演一位已飞升长生者的追随者，协助他完成更高的功业——实际上是让导师升为具名者。有三个使徒遗产：

| 遗产 | 导师 | 目标 | 相关司辰 |
|------|------|------|---------|
| **铸之使徒** (Forge Apostle) | 塑形者 (Shaper) | 大功业——建造破晓器核心，带来"拂晓" | 白日铸炉 |
| **杯之使徒** (Grail Apostle) | 愉悦者 (Delight) | 谢肉祭——最盛大的筵宴 | 赤杯 |
| **灯之使徒** (Lantern Apostle) | 沐光明者 (Illuminated One) | 收集七名见证者，将三瓣之门向外敞开至"第三态" | 守夜人 |

> **证据**: `"斯人抚育我们，造就我们；斯人在流放中等待，必将于我们带来拂晓之时再起。"`（`apostleforge.mentor`）

> **证据**: `"我们以之为甘露和欢乐之源者，斯人静待于放逐之途，直到最盛大的筵宴——'谢肉祭'来临。"`（`apostlegrail.mentor`）

> **证据**: `"我们的物质源头，精神源头，现居于漫宿者。"`（`apostlelantern.mentor`）

使徒本质上处于位阶体系的一个奇特位置：他们自己是凡人/信徒，但在协助一位长生者导师完成飞升（长生者晋升具名者）。他们充当导师与司辰之间的中介。例如，塑形者对其使徒说："我须得扮演铸炉"，表明长生者在使徒的协助下模仿司辰以完成最终的晋升。

#### c) 永生不死的敌人

特定的具名敌人，如 J·C、某些"藏形匿影的人物"（日记作者、裘里安、特里丰、韦兰），标注为 `["永生不死的敌人"]`：

> **证据**: `"他的势力让人望而生畏；他的身份，暂时不为人知。"`（元素 `diarist_unknown`）

这些敌人可能拥有"弱点"（Weakness），与特定的司辰影响力相关：

> **证据**: `"双角利斧不眷顾我的大敌。它的影子不论落在哪里，都会让他犹豫不前。"`（`rkx.weakness.horned`）

> **证据**: `"如果关于这东西的故事有一半是真实的，那么它以几乎同等的热情憎恨着一切。但对于我的大敌，狼的怒火燃烧得更为炽热。"`（`rkx.weakness.wolf`）

5. **孔雀之门** (Peacock Door) —— "又名眼瞳之门，或者自满之沮"
6. **三尖门** (Tricuspid Gate) —— 最终通往长生之境
