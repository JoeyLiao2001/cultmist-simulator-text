> 分析日期: 2026-07-02

## 一、概述：密教中文的核心美学

密教模拟器中文文本的美学核心是**克制的暗示**与**雅致的疏离**。它既非古文的佶屈聱牙，亦非现代口语的直白坦率——而是一种悬停在两者之间的书面语，借用有限的文言虚词、节制的意象密度与刻意回避的解释性语言，营造出 1920 年代学者笔记的质感：客观而不冷漠、惊悚而不张扬、诗意而不煽情。

## 二、句式模式（Sentence Patterns）

以下统计基于 1,950 条有效中文描述的全部文本内容。

### 2.1 核心句式（按出现频次）

#### 据说……（16 条，0.8%）
数据集中"据说"的所有出现均符合 CS 的间接引用传统——引述不确定的传闻、不可靠的记载或口头流传的秘密知识，而非陈述确定事实。

```
"据说第六印记出现的标记就是虎口间的皮肤可以拉到小指关节处……"
"先锋派编舞家尼古拉斯·凯尔创作的气势汹汹的芭蕾舞中据说编入了此密续。"
"据说，某种特定的知识只能通过特定种类的沉默来表达。"
"据说它被铸造出来，就是为了佩于王者之身，在未知的季节对抗一座尚未落成的都市。"
```

用法要点：'据说'后不加'是'形成判断句，保持语焉不详的暧昧。优先用于密传、工具和书籍类文本。

#### ……已被……（7 条，0.4%）
全部用于表示某个事物被外力改变至不可逆状态——被耗尽、被曝光、被抹除、被剥夺、被消灭、被熏黑。

```
"已被耗尽之物。"
"然而本书被去掉了大量书页；出版者的名字遭人涂黑；封面上还钉着一张语意不祥的纸条告知读者，威利斯福特已被'抹除'。"
"有一位司掌纪念与哀悼的神灵，他已被剥夺至尽，然而他不受欺骗。"
```

用法要点：以被动语态暗示超出凡人控制范围的力量在起作用。适合冬、铸、刃相文本。

#### ……与……相……（5 条，0.3%）
文言风格的比较/关联结构，用于物件外观描述和抽象概念间的类比。

```
"硕大的碗状物，尺寸与锣相当，由黑色的玄武岩雕刻而成……"
"……是与这门颇具挑战性的语言短兵相接时的必备武器。"
```

#### ……亦是……（1 条正式出现，但变体"便是"8 条、"乃是"、'即是'常见）
判断句式在 CS 中文中极其常见。"便是"出现 8 次，"乃"系判断（乃、乃是）频繁用于定义性陈述——秘传的核心教义常用此模式。

```
"伤口、门槛和揭示皆乃门的面相。"
"守夜者不入眠，入眠者不守夜。"
```

#### 无人知晓（0 条）
出乎意料——数据集不含"无人知晓"的精确匹配。CS 中文处理"未知"的方式更委婉：用"秘"（124 条, 6.4%）、"隐"（47 条, 2.4%）、"据说"、"被遗忘"等词汇替代直接的"无人知晓"陈述。这一发现本身就是重要的风格约束：**不要直接说出"无人知道"，用暗示的手法表现不可知**。

### 2.2 高频修辞结构

| 结构 | 出现次数 | 用途 |
|------|---------|------|
| `并非` | 8 | 否定性定义——先破后立 |
| `便是` | 8 | 判断句——定义本质 |
| `从不` | 4 | 否定副词——表现永恒特质 |
| `唯有` | 出现于工具与冬相文本 | 排他性陈述 |
| `倘若` | 出现于条件句 | 假设性语气 |
| `无论` | 5 | 让步结构 |
| `亦` | 11 | 并列补充——文言连词 |

### 2.3 句式设计原则

1. **长短交替**: 经典的 CS 段落通常由 1-2 句短句（开篇） + 1 句中长句（展开）组成。短句定调，长句丰富细节。
2. **避免连续三个以上短句**: 会导致破碎感，偏离学者笔记的沉稳感。
3. **优先使用句号**: 感叹号几乎不出现在正文中（仅在召唤物的台词中使用），问号罕见。所有句子应以句号结束。
4. **被动优于主动**: "据说"、"已被"、"被发现"、"被视为"——被动语态是 CS 的基本语法姿态，暗示信息是被接收的而非被创造的。
5. **判断句优于叙述句**: "此即……"、"乃是……"、"便是……"——定义式的口吻比讲故事的口吻更符合秘传文本的质感。

## 三、词汇层级体系（Vocabulary Tiers）

### 3.1 文言虚词频率

基于 1,950 条描述的全部文本统计（总出现次数）：

| 虚词 | 总出现次数 | 每描述平均数 | 主要用法 |
|------|-----------|-------------|---------|
| 之 | 503 | 0.26 | 领属标记、代词（"其文如下"、"无形之术"） |
| 为 | 381 | 0.20 | 判断词（"是为……"）、被动标记、介词 |
| 者 | 329 | 0.17 | 名词化（"逝者"、"具名者"）、话题标记 |
| 于 | 259 | 0.13 | 处所介词（"于林地"、"于十字路口"） |
| 斯 | 198 | 0.10 | *注意：绝大部分出现在人名/地名音译中（帕斯卡、克里斯托弗），非古典用法* |
| 所 | 183 | 0.09 | 被动标记（"为……所……"结构）、处所 |
| 其 | 181 | 0.09 | 代词（第三人称/远指）、领属 |
| 此 | 156 | 0.08 | 近指代词（"此即"、"此书"、"此物"） |

**使用约束**:
- **之**: 作领属标记时每段不超过 2 次（如"灯之准则"、"冬之静默"）。作代词时（"其文如下"）更克制——仅在引用典籍时使用。
- **者**: 主要用于术语构成（"具名者"、"敬拜者"、"观者"），少用于句中停顿。
- **于**: 用在处所和时间（"于林地"、"于午夜"），不用于抽象比较（不用"重于生命"，用"比生命更重"）。
- **此/其**: 交替使用以避免重复。"此"倾向用于近距离指称（手中的物品、眼前的景象），"其"用于远距离或抽象指称。
- **所**: 几乎全部出现在"为……所……"被动结构中（"为恐惧所困"、"为光芒所照耀"）。不单独用作处所名词。
- **斯**: 避免使用。它不是 CS 中文的有效虚词——所有出现都是音译副产品。

## 四、常见错误与反面模式（Common Mistakes to Avoid）

以下错误模式均在数据集中实际存在——它们是游戏内文本的特征，但**不应在 OC 创作中模仿**：

### 4.1 解释性语言

**错误症状**: 告诉读者某件事意味着什么，而非展示它是什么。

**数据集中的实际案例**（OC 创作应避免）:
```
"我知道这就是第三印记。"  ← 直接解释，消解神秘感
"我感到我的骨头也开始松弛瓦解了。" ← 我 + 感到 = 解释框架
```

**正确做法**（数据集内的正确案例）:
```
"据说第六印记出现的标记就是虎口间的皮肤可以拉到小指关节处。"
——观察式描述，让读者自己做判断。
```

### 4.2 现代口语化表述

**数据集中出现的问题模式及频次**:

| 问题词汇 | 出现次数 | 说明 |
|----------|---------|------|
| 真的 | 7 | "传言是真的"——口语化强调 |
| 非常 | 13 | "我的皮肤已经变得非常非常松弛了"——重复使用尤劣 |
| 完全 | 14 | 绝对的语气削弱暗示空间 |
| 特别 | 9 | 口语化副词 |
| 极其 | 10 | 痕迹在游戏指令文本中（"极其详尽的文章"）；文学性文本中应避免 |
| 事实上/实际 | 2 | 解释性过渡词 |
| 大概 | 18 | 口语化模糊词；应使用"或许"（44次）或"也许"（30次）替代 |

**正确做法**:
- 不用"非常" → 用意象承载程度（不写"非常冷"，写"空气冻得皮肤发脆"）
- 不用"真的" → 省略（不写"传言是真的"，写"传言属实"或直接陈述事实）
- 不用"完全" → 用"尽"、"殆尽"或省略
- 不用"极其" → 在文学性文本中以意象传达极端程度

### 4.3 过度修饰

**错误症状**: 多个形容词堆叠修饰同一名词。

**数据集中的负面示例**:
```
"一篇耸人听闻的花边文章，描述了关于'隐秘世界'的一知半解的传闻和当今道德观如何脱节……"
```
此句包含 5 个以上修饰成分——信息密度高但方式冗余。

**正确做法**: 数据集中最好的描述通常只有 1-2 个修饰词，更多依赖名词和动词的精选而非形容词的堆砌。
```
"空气冻得皮肤发脆。霜结上了玻璃。" ← 零形容词，全以动词驱动意象。
```

### 4.4 情绪化描写代替克制暗示

**错误症状**: 直接描写人物的情绪反应，而非以外部细节和环境变化暗示情绪。

**数据集中的问题案例**:
```
"不知名的恐惧正在用利齿啃噬着我的希望；一种关于存在本身的恐惧。"
```
此句直接命名了"恐惧"和"希望"——已接近情绪描写的上限。更好的做法是完全不命名情绪。

**正确做法**:
```
"光透过裂缝漏入。我的头脑比任何时候都更清晰。我升得越高，见得越多。"
——入迷状态的描写，未使用任何情绪词汇。
```

### 4.5 因果连接词滥用

**数据集中出现频次**:

| 连接词 | 出现次数 | 风险 |
|--------|---------|------|
| 因为 | 7 | 破坏神秘感的头号元凶 |
| 所以 | 5 | 同上 |
| 因此 | 6 | 同上 |
| 于是 | 3 | 叙事化过渡——CS 不应"讲故事" |

**核心原则**: CS 的中文描述是**现象学**的，而非**解释性**的。它描述所见、所闻、所感，但不追溯原因。每一个"因为"都在消减一分不可知性。

### 4.6 游戏指令语与文学描写混淆

数据集中 9.1% 的描述包含"请"、"需要"、"用于"、"放置"、"探索"等游戏指令标记。这些在游戏内是必要的 UI 语言，但在 OC 创作中应严格隔离：
- 卡牌的**文学描述**（引文）：不可使用指令语
- 卡牌的**游戏字段**（Aspects, Requirements, Effect）：可使用指令语

## 五、类别特定规范（Category-Specific Conventions）

### 5.1 书籍（Books）— 40 条

**典型长度**: 10-110 字，平均 49.3，中位数 47.0。

**结构特征**:
- 通常以书籍的编纂者、出版背景或历史地位开篇
- 中间段落简述内容或特色（但绝不概括全文）
- 常以该书的命运（被禁、失踪、仅存残本）收尾

**代表性文本**（中位数范例，47 字）:
```
莫兰小姐存货里的次要品目尚在。随着时间经过，有陌生人带着书误闯进来售卖，有时候我们会满足他们。
```

**创作规范**:
- 不要使用"本书介绍了……"的目录式开篇——太像课本
- 优先采用该书的命运（"据记载，她曾命人将这些秘密教义记录成书，以保其免于遗失……遗憾的是，据传说……"）或编纂者的神秘经历
- 引用书内的一句话比概括全书内容更有力量
- 可以注明语言、出版年月、作者信息作为质感锚点

### 5.2 追随者（Followers）— 38 条

**典型长度**: 2-108 字，平均 30.6，中位数 24.5。

**结构特征**:
- 极简人物速写：一个特征、一个习惯、一种不安的暗示
- 通常是外貌细节 + 一句暗示（而非直接说明）其本质的话

**代表性文本**（中位数范例，25 字）:
```
柴玛藏身于只有她自己知道的黑暗角落，正在舔舐伤口。
```

**创作规范**:
- 名称应为"称号 + 名字"格式（"理发师祖尔菲亚"、"走私商人乔吉奥"、"女猎手拉拉·柴玛"）
- 描述只展示一个人物切片——读者需要自己拼凑完整形象
- 第一人称独白仅用于特定追随者（多为杯子/蛾子追随者或召唤物）
- 优秀的人物描述让读者在脑中构建问题，而非传递答案

### 5.3 密传碎片（Fragments）— 88 条

**典型长度**: 2-83 字，平均 38.9，中位数 37.5。

**结构特征**:
- 常以一句教义核心作为引文（放在引号内），再跟一句评注或补充
- "引文 + 陈述"的双层结构是其标志模式
- 引文部分往往使用判断句式（"……乃……"、"……即是……"）

**代表性文本**（中位数范例，38 字）:
```
白日铸炉、弧月和昕旦是光源之神。汞、银和金乃功业的种穰，下文即其秘密教义……
```

**创作规范**:
- 每片碎片应围绕一句可被引用的核心语句构建
- 引文需有箴言/经文质感——可被单独摘出并仍然有力
- "下文即其秘密教义"是经典的后置铺垫——暗示存在更多但不提供
- 评注部分应简短（不超过引文长度的 1.5 倍）

### 5.4 工具与物品（Tools）— 69 条

**典型长度**: 9-116 字，平均 38.9，中位数 33.0。

**结构特征**:
- 外观描写极简（通常仅 1-2 个特征）
- 重点在物品的**起源传说**或**使用后果**，而非其功能说明
- 常有警告性语气

**代表性文本**（中位数范例，33 字）:
```
CAPUT GERAT LUPINUM。太阳曾被分裂，此即其伤口。
```

**创作规范**:
- 从不写"可用于……"——这是游戏说明，不是文学描述
- 以持有者（过去的或现在的）的感受或经历作为描述框架
- 外观特征应服务于 lore 暗示（如"猩红乃高热之色，而水银则是通往终极融解的钥匙"——颜色说明变成了宇宙论陈述）

### 5.5 灵体/召唤物（Spirits）— 46 条

**典型长度**: 2-103 字，平均 39.1，中位数 37.0。

**结构特征**:
- 常包含召唤物自身的直接引语（"噢，世界是平的……告诉我个秘密。"）
- 引语通常带有古怪的、不完全属于人类逻辑的质感
- 第三方叙述通常比追随者描写更少、更神秘

**代表性文本**（中位数范例，37 字）:
```
"噢，世界是平的；噢，可它是平的！我不会久留，我不该久留。告诉我个秘密。"
```

**创作规范**:
- 召唤物的台词应打破人类语法和逻辑习惯——句子可以拼接、跳跃、缺乏衔接
- 如果使用第三方描写，应比追随者更简练（召唤物的不可知性更高）
- 可包含重复、对声音而非内容的关注（"喀嚓喀嚓喀嚓"）

### 5.6 影响（Influences）— 54 条

**典型长度**: 1-55 字，平均 18.3，中位数 16.5。

**结构特征**:
- 最简练的类别——通常 1-3 句，描绘一种身心状态的体验
- 常使用破折号连接断裂的感知（"云朵分离——墙壁碾动——旧伤作痛。"）
- 从不直接命名情绪

**代表性文本**（中位数范例，17 字）:
```
云朵分离——墙壁碾动——旧伤作痛。
```

**创作规范**:
- 影响描述是 CS 中文最纯粹的展示——一行字必须传达完整的身心状态
- 使用外部世界变形的意象来映射内部状态（而不是说"我感到……"）
- 三个层级的影响应有递进：个体的 → 环境层的 → 宇宙论层的

### 5.7 地点（Locations）— 109 条

**典型长度**: 2-146 字，平均 55.2，中位数 53.0。

**结构特征**:
- 数据集中最长的平均描述长度
- 以丰富的地理/历史细节构建氛围
- 常融合具体的历史参考和超自然的暗示

**代表性文本**（中位数范例，53 字）:
```
奥里弗拉姆专卖古玩，香水，生物标本和稀有文献。自被一场无法解释的大火烧毁后，如今，它于局促的新址展开业务。
```

**创作规范**:
- 地点是唯一可接受更大篇幅描述的类别——地点需要空间建立"现实感"作为超自然元素的对立基础
- 具体的地名、街道名、人物名增加质感（但不是虚构的百科全书式堆砌）
- 将超自然元素放置在一句末尾或段落末尾——作为一个低沉的回声，而非响亮的主调

### 5.8 漫宿/梦境（Mansus）— 29 条

**典型长度**: 11-133 字，平均 43.0，中位数 38.0。

**结构特征**:
- 常引用具体书目（"特蕾莎·加尔米耶在书中考察了……"）
- 引用学术框架来框架化超现实体验

**代表性文本**（中位数范例，38 字）:
```
特蕾莎·加尔米耶在书中考察了匠人们所经历的神秘梦境中的相似之处。丛书第一卷。
```

**创作规范**:
- 漫宿文本是 CS 中最接近"学术引用"的类别——常以引用某位权威的著作为框架
- 语气应比地点描写更超然，但比密传碎片更结构化
- 梦境元素应保持逻辑性的表面——让读者自己发现其中的荒谬

### 6.2 长度与内容类型的关系

**短描述（10-20 字）**：
适用: 影响、追随者速写、极简工具描述
风格: 一个意象、一条暗示、一种情绪——不铺陈
典型:
```
"祖尔菲亚的剪刀从不离身。"  ← 9 字，追随者
"云朵分离——墙壁碾动——旧伤作痛。" ← 17 字，影响
```

**中等描述（25-50 字）**：
适用: 密传碎片、书籍摘要、灵体、原料
风格: 1-2 句 + 一句暗示性收尾
典型:
```
"白日铸炉、弧月和昕旦是光源之神。汞、银和金乃功业的种穰。" ← 33 字，密传
```

**长描述（50-90 字）**：
适用: 地点、漫宿、部分工具和书籍
风格: 地理/历史细节 + 神秘暗示作为第二层
典型:
```
"奥里弗拉姆专卖古玩，香水，生物标本和稀有文献。自被一场无法解释的大火烧毁后，如今，它于局促的新址展开业务。" ← 53 字，地点
```

**超长描述（90+ 字）**：
仅适用: 特殊地点、漫宿长文本
出现率: 仅 2.2% 的描述超过 100 字
约束: 必须有足够丰富的地理和历史细节作为支撑，而非空泛的修辞堆砌

### 6.3 长度设计的核心原则

1. **影响、追随者始终极简**: 它们依赖速写的力量，而非详细的描摹
2. **地点可最长**: 它是唯一的"读者需要找到自己在何处"的类别——空间描写的需求是正当的
3. **碎片遵循"核心引文 + 一句评注"的双层结构**: 这意味着其长度由引文长度决定，通常应控制在 25-60 字
4. **书籍不应变成目录条目**: 如果书籍描述超过 80 字，检查是否有解释性语言可以删去
5. **OC 创作建议对标游戏原有的"描述性"类别（而非"游戏指令"类别）的长度分布**: 即中位数约在 30-55 字之间

## 七、叙事视角（Narrative Voice）

### 7.1 数据集中的视角分布

| 视角 | 占比 | 典型场景 |
|------|------|---------|
| 完全非人称（无任何人称代词） | 43.0% | 密传碎片、冬之文本、宇宙论陈述 |
| 第一人称（"我"） | 33.1% | 影响、追随者（限特定角色）、启明/感官诱惑路径 |
| 第三人称（"他/她/它"） | 33.5% | 追随者、书籍评注、地点叙事 |

注：百分比之和超过 100% 是因为同一个描述可以同时包含第一和第三人称。

### 7.2 视角选择原则

- **非人称优先**: 当不确定时，去掉人称代词。CS 最有力的描述往往不标记观察者。
- **第一人称用于个人体验**: 影响、诱惑路径、特定追随者的独白。使用第一人称时，声音必须具有个体性——不能是泛化的"我"，而是一个具体的、有古怪癖好的叙事者。
- **第三人称用于外部观察**: 追随者描写、书籍评注。叙述语气应保持学者的超然——即便描述的内容再恐怖。
- **禁止第二人称**: 数据集不含"你"字面的文学描述——这是一个被严格遵守的边界。

## 九、创作质量自检清单

使用本清单在实际创作过程中验证每条新生成的卡牌描述：

### 语言层
- [ ] 描述长度是否在类别推荐范围内？（参见第六章）
- [ ] 是否避免了"因为/所以/因此"等因果连接词？
- [ ] 是否避免了"非常/真的/完全/极其"等口语副词？
- [ ] 文言虚词（之/者/其/于）使用是否克制（每段不超过 3 个不同虚词）？
- [ ] 是否避免了直接称呼读者（"你"）？

### 内容层
- [ ] 是否暗示多于明示？（读者读后是否产生了更多问题而非得到了答案？）
- [ ] 是否包含至少一个 lore 锚点（司辰、准则、漫宿区域、历史事件）？
- [ ] 是否避免了"本书介绍了……"/"可用于……"/"这是一种……"等解释性框架？

### 风格层
- [ ] 去掉所有命名标签后，读起来是否仍然像 CS 文本？
- [ ] 情绪是否通过外部意象传达而非直接命名？
- [ ] 句子长短是否有交替（短句开篇 + 中长句展开）？
- [ ] 如果使用第一人称，叙事者的声音是否有特定质感（而非泛化）？

### 类别特定
- [ ] 书籍：是否避免了目录式概括？是否包含了编纂者/历史命运？
- [ ] 追随者：是否只是一个人物切片？是否特征鲜明而解释克制？
- [ ] 碎片：是否有可独立引用的核心语句？评注是否简短？
- [ ] 工具：是否以起源传说/使用后果代替了功能说明？
- [ ] 影响：是否以外部世界变形反映内部状态？是否在 30 字以内？
- [ ] 地点：是否有足够的具体细节支撑其稍长的篇幅？

## 十、参考示例: 数据集中各风格的范式文本

### 极简暗示（追随者速写范式）
```
"祖尔菲亚的剪刀从不离身。"
```
9 字。一个细节、一个"从不"——永久特质的标注。零解释。

### 宇宙论陈述（密传碎片范式）
```
"伤口、门槛和揭示皆乃门的面相。"
```
15 字。判断句。定义性口吻。X、Y 和 Z 的三项并列——三连音是 CS 密传文本的经典修辞。

### 现象学描写（影响范式）
```
"空气冻得皮肤发脆。霜结上了玻璃。"
```
16 字。两短句。不写"寒冷"，写皮肤和玻璃的物理反应。

### 知识考古（书籍范式）
```
"1860 年代中期，某位'伊女士'曾展示过一种娱乐方式，她称其源头来自本书。"
```
37 字。不概括书的内容。指出其历史中的一次出现。用引号标记一个模糊的身份。

### 叠层描绘（地点范式）
```
"奥里弗拉姆专卖古玩，香水，生物标本和稀有文献。自被一场无法解释的大火烧毁后，如今，它于局促的新址展开业务。"
```
53 字。第一句：具体商品清单（构建现实感）。第二句：无法解释的大火（神秘感的插入）。第三句：现状（"局促"——衰退的暗示）。
