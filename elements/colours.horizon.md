# 天际之色
> `colours.horizon`

> 在雾气散去，太阳或被误认为是月亮之时，于天幕上的某些斑块可见一种黯淡的白金色。我们屏住呼吸，看着它转亮，直到每一重颜色与临近的分离开来，仿若一个个新铸的文字。逆孵之卵。
